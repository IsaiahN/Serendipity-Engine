# Serendipity Engine PWA — Settings Page Wireframe

*A unified settings page that consolidates all user-configurable options scattered across the application. Accessible from a gear icon in the workspace top bar or via `Ctrl/Cmd + ,`.*

---

## Settings Navigation

The Settings page uses a **left sidebar navigation + right content panel** layout — the same pattern as VS Code, Figma, or Discord settings. Each category is a clickable section in the sidebar. The content panel shows the settings for the selected category.

```
┌──────────────────────────────────────────────────────────────────┐
│  ⚙ Settings                                          [✕ Close]  │
├──────────────┬───────────────────────────────────────────────────┤
│              │                                                   │
│  General     │  ┌─────────────────────────────────────────────┐  │
│  AI Models   │  │                                             │  │
│  Workspace   │  │  (Content for selected category)            │  │
│  Writing     │  │                                             │  │
│  Editor      │  │                                             │  │
│  Privacy     │  │                                             │  │
│  About       │  │                                             │  │
│              │  └─────────────────────────────────────────────┘  │
│              │                                                   │
└──────────────┴───────────────────────────────────────────────────┘
```

## Entry Points to Settings

Settings is accessible from:

1. **Gear icon** in the workspace top bar (always visible)
2. **Keyboard shortcut** `Ctrl/Cmd + ,` (standard settings shortcut)
3. **Command Palette** (`Ctrl/Cmd + K` → "Settings" or any setting name)
4. **Contextual links** — various parts of the UI link directly to the relevant settings category:
   - "Change model" links in chat headers → AI Models
   - "Adjust" links in Teaching Tips → Writing > Teaching & Guidance
   - "TTS settings" in the bottom bar → Writing > TTS
   - "View writing profile" in author.md → Writing Profile
   - "Configure Editor" in Phase 9 → Editor

Settings changes take effect immediately (no save button needed). A brief toast confirms changes: "Setting updated."
