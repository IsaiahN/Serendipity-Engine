> **Roll freely across this list.** These names are drawn from many cultural traditions — do not pre-filter by assumed cultural "match" before rolling. Cultures mix; people carry names from traditions they don't fully inhabit; naming is itself a story. Roll first, then ask: *why does this character have this name?* That question often opens backstory and world detail.
>
> **Exception — monoculture worlds:** If the story's world is explicitly monocultural by deliberate design (a closed community, isolated region, or homogeneous society), you may pre-filter to names from the relevant tradition before rolling. Document this constraint in `world/world-building.md`. See `Story/World Building/questions.md` — Cultural Composition section.
>
> ⛔ **Forbidden name — "Claude" must never be used as a character name, place name, object name, or any other named entity in any story generated by this engine.** The name conflicts with the AI assistant used to run the engine.
